# Plugin Medical

> Esse repositório contém uma extensão recuperada do jogo Unturned. A desenvolvi há muito tempo atrás, por volta de 2023, infelizmente perdi os arquivos originais devido a um SSD defeituoso, no momento da publicação ainda se encontra funcional e o código disponibilizado foi obtido a partir da descompilação da DLL utilizando o DotPeek, portanto releve as redundâncias e formatação do código.

## ⚠️ Aviso

O projeto foi desenvolvido unicamente com fins recreativos, não há previsão para atualizações, futuros reparos nem qualquer garantia de qualidade ou segurança! Utilize a aplicação por sua conta e risco.
